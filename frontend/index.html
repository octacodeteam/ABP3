Sobre o Projeto

O GeoInsight foi desenvolvido para permitir que agricultores e pesquisadores visualizem e comparem dados geoespaciais gratuitos de satélites. O objetivo é oferecer uma ferramenta simples e acessível que facilite a análise de dados como NDVI, umidade do solo e temperatura da superfície, apoiando a tomada de decisão no setor agrícola.

Como Funciona o Mapa

Siga estes passos para explorar dados geoespaciais de forma simples:

1. Escolha um Local

Você pode clicar diretamente em qualquer ponto no mapa ou usar a barra "Pesquisar por Local" na lateral para encontrar uma cidade ou região específica.

2. Veja os Dados Disponíveis

Após selecionar um local, a barra lateral direita mostrará automaticamente todos os dados de satélite (como Sentinel, Landsat) disponíveis para aquele ponto.

3. Filtre seus Resultados

Use os filtros na barra esquerda para refinar sua busca por "Coleção" ou por um período específico usando a "Data de Início" e "Data de Fim".

4. Compare os Dados

Marque a caixa de seleção ao lado de dois ou mais resultados e clique em "Comparar Selecionados" para ver um comparativo detalhado das informações.
